Installation

Whose News runs on a variety of different platforms, so you can just pick one and install it.

Chrome Extension

The Chrome Extension is fully featured, which allows you to see who owns the site you read at a glance, among with many other more amazing parts of Whose News.

Chrome Extensionclose

To install Whose News for Chrome, just click the button below:

Install

To install Whose News for Chrome, open this page in Chrome.

Install NowChrome Webstore
Bookmarklet

The bookmarklet lets you use Whose News without installing an extension to your browser. It's not as fully featured as an extension, but can work on any browser that supports modern Javascript.

Bookmarkletclose

To install the Whose News bookmarklet, just drag the icon below onto your browser's bookmarks bar.

Install Now
Node

Whose News for node.js allows developers to use our data to create anything. They could create a new interface, support a new browser, or do something no-one has thought of yet.

NPMclose

To install Whose News, install whose-news via npm.

npm install --save whose-news
